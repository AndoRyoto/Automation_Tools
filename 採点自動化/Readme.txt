ＴＡの仕事の一つ成績まとめ、各講義の提出物をまとめるのが面倒くさい。
履修者の課題提出回数をリストでまとめるbatファイルを作成
（実は欠席者を数えるほうが多分早い、半分効率化半分好奇心でやってます）

使い方
・各回から提出物をダウンロードし、同じディレクトリに解凍する
・評定->ユーザレポートをエクセルスプレッドシートでエクスポート（オプションは何でもよい）
・エクスポートしたファイルを名前を付けて保存->CSV、UTF-8、カンマ区切りのcsvファイル、
　文字コードをANSIにして（重要）、ファイル名list.csvで保存
・同ディレクトリにmakelist.batを準備
・makelist.batを実行
test1.csv = 登録済みの生徒の学籍番号リスト
test2.csv = すべての回で課題を提出した生徒を重複してのせた学籍番号リスト
test3.csv = 登録済みの生徒の学籍番号と提出回数のリスト

・test3.csvをエクセルにでも張ってあとは頑張る。
・各回の提出状況はやってません。
　test2.csvに回ごとに境目を作って照合のとき何回かの変数を出力すればいけそう
